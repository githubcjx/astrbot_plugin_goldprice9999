# astrbot_plugin_goldprice

查询**黄金9999（Au99.99）**实时金价的 AstrBot 插件。在群里发送触发指令（默认 `/金价`），插件会拉取实时行情并把结果回复到群聊。

## 功能

- 发送 `/金价` 即返回黄金9999的最新价、涨跌、涨跌幅、开盘/昨收/最高/最低、买卖盘口。
- **可配置触发指令**：默认 `/金价`，可在插件配置里改成任意文本。
- **可配置生效群聊**：可指定只在某些群生效；留空则所有群都生效，私聊始终生效。

返回示例：

```
📊 黄金9999 实时行情
最新价：892.00 元/克 🟢
涨跌：-23.63（-2.58%）
今开：915.00　昨收：915.63
最高：915.00　最低：887.01
买价：891.50　卖价：892.00
更新：2026-06-11 15:30:21
```

## 安装

把整个 `astrbot_plugin_goldprice` 目录放到 AstrBot 的 `data/plugins/` 下，重启或在 WebUI 里重载插件。依赖（`aiohttp`、`pycryptodome`）会由 AstrBot 自动安装。

## 配置项

在 AstrBot WebUI 的插件配置页设置：

| 配置项 | 说明 | 默认 |
| --- | --- | --- |
| `trigger_prefix` | 触发指令（精确匹配），群成员发送此内容即查询 | `/金价` |
| `enabled_groups` | 生效的群聊 ID 列表，留空表示全部群聊生效 | `[]` |

> 提示：若你的平台会吞掉开头的 `/`，把 `trigger_prefix` 改成 `金价` 即可。

## 数据来源与实现说明

数据来自融通金 H5 行情页（`i.jzj9999.com`）所用的实时行情网关：

- 协议：WebSocket + Protobuf 二进制，地址 `wss://rtjwbqt.ytj9999.com:8443/gateway`。
- 鉴权：连接后发送 Blowfish/CBC 加密的 token，鉴权成功后订阅合约 `Au99.99`，网关推送实时行情。
- 本插件内置了一个仅覆盖本场景字段的极简 protobuf 编解码器，**不依赖** protobuf 运行库；加密用 `pycryptodome` 的标准 Blowfish（已校验与服务端一致，插件加载时会自检）。

该接口为第三方非公开接口，可能随对方前端更新而变化；如失效需重新对照其前端 JS 调整常量与字段。仅供学习交流，请勿用于商业用途或高频请求。
